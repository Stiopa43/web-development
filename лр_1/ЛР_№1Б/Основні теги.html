<body> - тег для створення тіла веб-сторінки (містить тільки теги, які відповідають за контент)
<head> - тег для створення основи веб-сторінки (містить тільки мета теги)
<font> - тег, який призначений для роботи із текстом (щоб задати йому конкретні стилі за допомогою атрибутів)
<hr> - тег, який призначений для створення лінії
<img> - тег, який призначений для вставлення зображення
<p> - тег для створення нового параграфу
<marquee> - тег для створення області із рухомим текстом
<br> - тег для переносу текста на інший рядок
<b> - тег для збільшення жирності шрифта
<title> - тег, щоб задати заголовок веб-сторінки
<html> - основний тег для заключення всіх інших
